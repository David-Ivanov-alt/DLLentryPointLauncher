# DLLentryPointLauncher
This program can launch entry points from *.dll and *.cpl files.                                                  
Эта программа может запускать точки входа из *.dll и *.cpl файлов.

If antivirus complains about an main.exe file, add the certificate. Here's how:                                                            
Если антивирус ругается на main.ехе файл то добавьте сертификат, вот как это сделать:

Open Properties -> Digital Signatures, select the first option in the list, and click Details.                                                     
Откройте свойства -> цифровые подписи, в списке выберете MyEntryPointLauncher, и нажмите сведения.

Next, click "view certificate".                                                                                     
Далее нажмите "просмотр сертификата".

And click "install certificate...".                                                                               
И нажмите "установить сертификат...".

In the window that opens, select "local computer" or "current user" and click "Next".                                              
В открывшемся окне выберете "локальный компьютер" или "текущий пользователь" и нажмите "Далее".

Then click next again.                                                                                   
После снова нажмите далее.

And click "Done!".                                                                       
И нажмите "Готво!".

Afterwards, close all windows by clicking "OK".                                                                   
После закройте все окна нажатием на "ОК".
